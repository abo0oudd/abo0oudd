# قالب رابط السيرة المجاني • Free Link-in-Bio Template

**قالب أنيق وفخم، يحتوي على كريبر 3D تفاعلي، جاهز للتخصيص والنشر المجاني.**

## 📦 الاستخدام
1. حمل أو انسخ ملف `index.html`.
2. غيّر البيانات في الأماكن المحددة (الصورة، الاسم، الروابط).
3. ارفعه على أي مستضيف مجاني (GitHub Pages, Netlify, Vercel).

## 🎨 المميزات
- تصميم داكن احترافي.
- متجاوب مع الجوال.
- مجسم كريبر 3D من Sketchfab.
- أيقونات Font Awesome 6.
- خطوط Inter العصرية.

## 🔧 التخصيص
كل ما تحتاج تغييره موجود في الأعلى داخل `<!-- ========== ابدأ التخصيص ========== -->`.  
بدّل الروابط والنصوص والصورة، وخلال دقائق يكون موقعك جاهز.

## 📄 الترخيص
هذا القالب **مجاني تماماً**، يمكنك استخدامه وتعديله ونشره بدون قيود.
