# موقع vilaosk الشخصي - رابط السيرة

قالب رابط سيرة ذاتية مميز، يحتوي على صفحتين (الرئيسية + مواقع التواصل) باللغتين العربية والإنجليزية.

## المميزات
- 🌐 لغتين: عربي (RTL) وإنجليزي (LTR)
- 🎨 تصميم داكن مع لمسات نيون حمراء وتأثيرات زجاجية
- 📱 متجاوب مع الجوال والتابلت والكمبيوتر
- 🔢 عداد زوار حقيقي
- 🚀 سهل الرفع على GitHub Pages, Netlify, Vercel
